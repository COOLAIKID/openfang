## SECTION 2: Five Bullet Points

**Character limit per bullet:** 200 characters (some categories allow up to 500)

**Rules for bullet points:**
- Start each bullet with an ALL CAPS feature name followed by a dash: "FEATURE NAME — benefit statement"
- Lead with the most important feature/benefit in Bullet 1
- Each bullet should address a different buyer concern or benefit category
- Include secondary keywords naturally across all 5 bullets
- Do NOT use pricing, availability, shipping info (Amazon prohibits this)
- Do NOT use seller-specific information in bullets
- Each bullet = one feature + one benefit + one proof point or use case

**Bullet structure template:**
FEATURE NAME — [What it does] so that [who can] [achieve outcome]. [Optional: proof or specific detail].

Write all 5 bullets in order of priority:

**Bullet 1 (Primary Benefit — most compelling):**

**Bullet 2 (Key Differentiator — how you're different from competitors):**

**Bullet 3 (Use Case / Versatility or Reliability):**

**Bullet 4 (Quality / Materials / Durability):**

**Bullet 5 (Risk Reversal / What's Included / Guarantee):**

---

## SECTION 3: Product Description (A+ Content Ready)

**For standard listings:** 2000 characters max
**For A+ Content / Brand Registry:** Write in modules

Write a 300-word product description that:
- Does NOT repeat the bullets verbatim
- Tells the product story — who it's for and why it exists
- Paints a picture of the customer's life with and without this product
- Includes the primary keyword in the first 100 words
- Uses short paragraphs (2-3 sentences each)
- Ends with a clear "Add to Cart" encouragement

If Brand Registry: suggest A+ Content module structure:
- Module 1: Comparison chart (vs. generic competitors)
- Module 2: Feature image with overlay text
- Module 3: Lifestyle image caption
- Module 4: Story/brand module

---

## SECTION 4: Backend Search Terms

**Character limit:** 250 bytes total (including spaces counts toward limit)

Provide a list of backend search keywords. Rules:
- Do NOT repeat words already in the title or bullets
- Do NOT include competitor brand names
- Do NOT include ASINs
- DO include misspellings buyers commonly make
- DO include synonyms, alternate phrasings, and long-tail variations
- DO include use-case keywords ("gift for dad", "office desk", "camping gear")
- Separate with spaces, NOT commas (Amazon doesn't use commas as separators)
